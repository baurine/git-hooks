# Githook

A ruby gem that help to setup git hooks easily, base on Rake, inspired from Capistrano.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'git-hook'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install git-hook

## Usage

    $ githook install
    $ githook setup
